# Sales & Revenue Analysis Dashboard

## Objective
Analyze sales and revenue trends using interactive dashboarding.

## Tools Used
- Power BI
- Excel / CSV

## Features
- Total Sales KPI
- Total Profit KPI
- Sales Trend Analysis
- Category-wise Sales
- Region Filtering
- Product Performance

## Insights
- Technology category generated strong revenue.
- Sales varied across regions.
- Top products contributed significantly.
